# Openclaw Ollama Setup Website

A minimalist, dark-themed bilingual (English/Myanmar) frontend built with NestJS static serving.

## Features

- Beginner → Advanced sections for Openclaw + Ollama setup
- English / Burmese language toggle
- Rich dark UI with glassmorphism and motion effects
- Developer service/contact section

## Run

```bash
npm install
npm run start:dev
```

Then open `http://localhost:3000`.
