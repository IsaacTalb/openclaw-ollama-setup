# Openclaw Ollama Setup Website

A minimalist, dark-themed bilingual (English/Myanmar) frontend built with NestJS static serving.

## Features

- Beginner → Advanced roadmap for Openclaw + Ollama setup
- Mega Cheatsheet highlights section based on printable reference commands
- English / Burmese language toggle
- Rich dark UI with glassmorphism and motion effects
- Developer service/contact links to Isaac Talb portfolio

## Run

```bash
npm install
npm run start:dev
```

Then open `http://localhost:3000`.
